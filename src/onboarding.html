About BiasBuster

BiasBuster is a Chrome extension designed to help you cut through media bias and get to the facts. It analyzes news articles for bias, fact-checks claims, and rewrites content to present a more balanced perspective.

In today's polarized media landscape, it's increasingly difficult to find objective news coverage. BiasBuster helps by:

Getting Started

Using BiasBuster is easy! Follow these simple steps to start getting more balanced news:

1

Visit a News Article

Navigate to any news article on major news sites like NYTimes, WSJ, CNN, Fox News, etc.

2

Click the BiasBuster Icon

Click the BiasBuster icon in your browser toolbar or use the floating button on the page

3

Analyze the Article

Click "Analyze Article" to have BiasBuster analyze the content for bias

4

Read the Balanced Version

Review the rewritten article, bias analysis, and sources for a more complete picture

Setting Up Your AI Service

BiasBuster uses AI to analyze and rewrite articles. You'll need to provide your own API key from one of the following services:

OpenAI (GPT-4)

OpenAI's GPT-4 provides excellent analysis and rewriting capabilities.

  1. Visit OpenAI to create an account
  2. Navigate to the API section and create an API key
  3. Copy your API key
  4. Open BiasBuster settings and paste your API key in the appropriate field
  5. Select "OpenAI" as your AI model

Anthropic (Claude)

Anthropic's Claude is known for its thoughtful and nuanced analysis.

  1. Visit Anthropic to create an account
  2. Generate an API key from your account dashboard
  3. Copy your API key
  4. Open BiasBuster settings and paste your API key in the appropriate field
  5. Select "Anthropic" as your AI model

Google (Gemini)

Google's Gemini model offers powerful analysis capabilities.

  1. Visit Google AI Studio to create an account
  2. Navigate to the API section and create an API key
  3. Copy your API key
  4. Open BiasBuster settings and paste your API key in the appropriate field
  5. Select "Gemini" as your AI model

Note: Your API key is stored locally on your device and is never sent to our servers. You are responsible for any usage charges from the AI service provider.
